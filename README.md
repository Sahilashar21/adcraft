# AdCraft Studio

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Overview

AdCraft Studio is a powerful tool for creating and managing marketing campaigns. It features an AI-powered caption generator that helps you craft the perfect marketing copy in seconds. The application is built with a modern tech stack, including Next.js, React, and Tailwind CSS, and features a sleek, user-friendly interface.

## Features

- **Campaign Management**: Create, edit, and delete marketing campaigns with ease.
- **AI Caption Generator**: Automatically generate engaging captions for your campaigns.
- **Credit System**: Manage your usage of the AI generator with a simple credit system.
- **Modern UI**: A completely redesigned user interface with a focus on usability and aesthetics.
- **Responsive Design**: The application is fully responsive and works great on all devices.

## Tech Stack

- [Next.js](https://nextjs.org/) - React framework for building server-side rendered and static websites.
- [React](https://reactjs.org/) - JavaScript library for building user interfaces.
- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for rapid UI development.
- [MongoDB](https://www.mongodb.com/) - A NoSQL database for storing campaign and caption data.
- [Framer Motion](https://www.framer.com/motion/) - A library for creating animations in React.
